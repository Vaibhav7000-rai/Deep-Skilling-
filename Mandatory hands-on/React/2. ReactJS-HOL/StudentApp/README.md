# StudentApp

This is a minimal React app (Vite) named StudentApp. It includes three components:

- `Home` — displays the message "Welcome to the Home page of Student Management Portal"
- `About` — displays the message "Welcome to the About page of Student Management Portal"
- `Contact` — displays the message "Welcome to the Contact page of Student Management Portal"

Quick start:

1. Install dependencies

```bash
cd "c:/Users/acer/OneDrive/Desktop/Deepskilling/Mandatory hands-on/React/2. ReactJS-HOL/StudentApp"
npm install
```

2. Run dev server

```bash
npm run dev
```

Open the URL printed by Vite (usually http://localhost:5173).
